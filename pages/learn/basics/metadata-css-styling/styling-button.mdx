## Styling the Button

To style the button component, you can use [CSS Modules](https://github.com/css-modules/css-modules). Here’s how.

Inside the `components` directory, create a file called `ButtonLink.module.css` and add the following CSS:

```css
.link {
  color: #fff;
  background: #0070f3;
  line-height: 2.5em;
  display: inline-block;
  text-decoration: none;
  padding: 0 2.5em;
  border-radius: 5px;
}
```

Now, this can be imported in `ButtonLink.js` as `styles`. You can then use `styles.link` for `className` to apply the CSS.

```jsx
import React from 'react';
import Link from 'next/link';
import styles from './ButtonLink.module.css';

const ButtonLink = ({ href, children }) => (
  <Link href={href}>
    <a className={styles.link}>{children}</a>
  </Link>
);

export default ButtonLink;
```

The CSS is now applied!
