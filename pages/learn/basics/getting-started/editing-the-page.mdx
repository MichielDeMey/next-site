## Editing the page

Let’s try editing this page.

- Open `pages/index.js` with your text editor.
- Find the text that says **“Welcome to”** under the `<h1>` tag and change it to **“Learn”**.
- Save the file.

Now, as soon as you save the file, the browser automatically reloads and updates the page with the new text:

<Image
  shadow
  alt="Learn Next.js"
  oversize={false}
  src="/static/images/learn/getting-started/learn-nextjs.png"
  width={1030 / 2}
  height={384 / 2}
/>

The Next.js development server (started with `npm run dev` or `yarn dev`) has the **Hot Reloading** feature which monitors file changes and automatically updates the browser.
